# Evidence Folder

This folder should contain:

## `/annotation_examples/` (3–5 screenshots)
Screenshots showing your Roboflow annotations.

## `/val_predictions/` (10 screenshots)
Validation set predictions from the trained model.

## `/new_predictions/` (5 screenshots)
Predictions on new (unseen) construction site images.

## How to capture
1. Run the training notebook in Colab
2. Right-click on displayed prediction images → "Save image as..."
3. Upload to this folder in your GitHub repo
